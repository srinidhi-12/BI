# Sales Analysis Dashboard - Power BI
This project analyzes sales performance across categories, sub-categories, regions, and customer segments using Power BI.

## Key Insights
- Identified top 5 performing states by sales
- Highlighted loss-making sub-categories
- Compared sales and profit across regions
- Analyzed segment-wise profitability

## KPIs Used
- Total Sales
- Total Profit
- Total Quantity
- Profit Margin %

## Tools Used
- Power BI
- DAX
- Data Modeling
- Conditional Formatting
